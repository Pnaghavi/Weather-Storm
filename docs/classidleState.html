Detailed Description

This class is inherited from the State class of Elma that represents the idle state of the weatherStormStateMachine class.

The idleState class represents one of the states used by the weatherStormStateMachine class. This was necessary because the Elam State class is abstract. None of the entry, during, and exit methods perform an essential task.
