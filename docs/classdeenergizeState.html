Detailed Description

This class is inherited from the State class of Elma that represents the deenergize state of the weatherStormStateMachine class.

The deenergizeState class represents one of the states used by the weatherStormStateMachine class. Most of the code is written in the entry method because once that method is done the taskCompleted will become true and the manager that is running the process which is running the weatherStormStateMachine will emit event for the next state
