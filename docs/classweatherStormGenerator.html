Detailed Description

This class is inherited from the Process class of Elma.

The weatherStormGenerator class creates a weatherStormStateMachine object. It also creates a Manager object to schedule, init, and start the weatherStormStateMachine process. This process waits for the tasks of state to be completed before emitting the event for the next state to start. This process will also pass the json object received from the server in the get data state to the energize and deenergize states.
