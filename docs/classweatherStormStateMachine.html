Detailed Description

This class is inherited from the StateMachine class of Elma.

The weatherStormStateMachine class has four states that are: idel state, get data state, energize state, and deenergize state. there are four transitions for this state machine that are: start, that changes idel state to get data state; energize, that changes get data state to energize state; deenergize, that changes energize state to deenergize state; and idle, that changes deenergize state to idle state.
